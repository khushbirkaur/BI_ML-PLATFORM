# BI_ML-PLATFORM

An end-to-end Business Intelligence and Machine Learning platform built with Python and Streamlit.

This application allows users to upload any dataset and automatically perform:

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Business Intelligence Dashboards
- Machine Learning Model Training
- Time Series Forecasting
- Automated Insights

## Features

- Upload CSV or Excel datasets
- Automatic data preprocessing
- Interactive visualizations using Plotly
- Multiple ML models (Random Forest, Logistic Regression, Decision Tree)
- Forecasting with time series models
- Insight generation for business decisions

## Tech Stack

Python  
Streamlit  
Pandas  
NumPy  
Scikit-learn  
Plotly
